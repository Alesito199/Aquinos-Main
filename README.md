## 💻 Instalación

### Opción 1: Servidor Local Simple

```bash
# Clonar el repositorio
git clone https://github.com/tuusuario/Aquinos-Main.git
cd Aquinos-Main

# Opción A: Python
python -m http.server 8000

# Opción B: Node.js (con http-server)
npx http-server -p 8000

# Opción C: PHP
php -S localhost:8000
```

Luego abre: `http://localhost:8000`

### Opción 2: Despliegue en Hosting

1. **Subir archivos** vía FTP/SFTP o panel de control
2. **Configurar dominio** apuntando a la carpeta raíz
3. **Verificar permisos** de lectura en todos los archivos
4. **SSL/HTTPS** recomendado para producción

### Opción 3: Netlify / Vercel (Recomendado)

```bash
# Netlify
netlify deploy --prod

# Vercel
vercel --prod
```
